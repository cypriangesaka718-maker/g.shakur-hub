# G.shAkuR Kicks & Jersey Hub — Website

Starter Next.js + Tailwind site for G.shAkuR Kicks & Jersey Hub.

Local dev
1. npm install
2. npm run dev
3. Open http://localhost:3000

Contact form
- This project uses Formspree for form handling. Sign up at https://formspree.io/, create a form and get the form ID.
- Replace FORM_ID in pages/contact.js (action URL: https://formspree.io/f/FORM_ID)

Deploy
- Push this repo to GitHub and connect it to Vercel (https://vercel.com/new). Vercel will detect Next.js and deploy automatically.

Images & logo
- Add images to /public and replace the hero/item placeholders.
